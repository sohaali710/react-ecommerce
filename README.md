# React Shopping Cart

App live demo :

https://react-shopping-cart777.netlify.app/


##### Using :
- React router
- Use context
- Axios
- Json server
- Local storage


##### It contains :
- Products list page with:
  - Add to cart
- Cart page with:
    - Increase or decrease the quantity of cart item
    - Remove product from cart
    - Cart data is saved into localStorage
- Page to show product details
